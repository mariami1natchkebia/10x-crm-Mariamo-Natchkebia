# 10X CRM

## About
10X CRM is a customer relationship management tool that helps businesses organize, track, and nurture their interactions with leads, customers, and other contacts. It acts as a central hub designed to track contacts, manage sales pipelines, automate communication, and improve customer service, allowing users to efficiently manage client information, activities, and communications.

## Features
* **Client Management:** Add, view, and filter clients dynamically based on status (Lead, Contacted, Won, Lost).
* **Advanced Search & Sorting:** Real-time search by name or company, with sorting options ranging from newest/oldest to deal value and alphabetical order.
* **Interactive Modal Popups:** Sleek modal forms for adding new clients with built-in validation.
* **Persistent Storage:** Client records are safely stored using browser localStorage alongside remote API requests.
* **Responsive Dashboard:** Clean UI with dark/light themes and custom color palettes.

## Tech Stack
* **HTML5 / Semantic Markup**
* **CSS3 / Flexbox / Grid**
* **JavaScript (ES6+ / Async-Await / Fetch API)**
* **DummyJSON API**

## How to Run
1. Clone or download the project repository to your local machine.
2. Open the project folder in a code editor (such as VS Code).
3. Use a local server extension (like **Live Server** in VS Code) to run `index.html`, or simply open `index.html` directly in your web browser.
